SISTEMAS OPERATIVOS.

Los sistemas operativos son el conjunto de programas de ejecución de software y hacen las veces de mediador
entre el hardware y el usuario, busca lograr un rendimiento óptimo del computador. Los sistemas operativos nos
guian en el proceso de instalación y uso de las aplicaciones que necesitamos o usamos en la maquina.
El sistema operativo se desarrolla de tal manera que sea posible implementar funciones nuevas u optimizarse
sin afectar ell funcionamiento del mismo.
El sistema operativo administrar el hardware, reparte los recursos en las aplicaciones, tambien permite ó
facilita  la comunicación entre dispositivos como el mouse, teclado, microfono, etc. Además de todo esto
tambien permite las conexiones de red.
Los sistemas operativos pueden dividerse en:
	Multitareas: es una caracteristica de los nuevos SO quew permite el uso de aplicaciones al tiempo
	trabajando en diferentes planos de relevancia, son demasiado comunes.
	Monotarea: SO que solo puede hacer una tarea a la vez.
	Monousuario: son sistemas que unicamente se administran por un usuario.
	Multiusuario: estos sistemas cunmplen las tareas de dos o mas usuarios a la vez.
La mayoria de los sistemas operativos poseen una interfaz grafica para que su funcionamiento ó funciones
sean mas intuitivas.
